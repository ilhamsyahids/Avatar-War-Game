[NAMING CONVENTIONS]
The ADT headers made are made as base
You can add more method if needed
But please make sure to follow this convention in method making
Thank you

Constant & Typedef:
	<Type><What>

Predikat:
	<Is/Can><Type><What>
	<Type><Has><What>

Prototype:
	<Type><Action>

Constructor:
	<Type>CreateEmpty
	<Type>Create
	<Type>Initialize

Destructor:
	<Type>Dealokasi

Selector:
	<Type><Attribute>

Other Methods:
	<Type><Action>

Number Element = NBElmt
Max Element = MaxElement

Element will always be written Element except in NBElmt
